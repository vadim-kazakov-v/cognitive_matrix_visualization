# Переработка пресетов для 3-слойных экранов

## Введение
В рамках задачи по переработке пресетов для 3-слойных экранов были внесены изменения в файл `/workspace/templates/index.html`, чтобы улучшить поддержку 3-слойных радиационных экранов с учетом необходимых Constraints.

## Основные изменения

### 1. Новый пресет "3-Layer Shield"
- Добавлена новая кнопка `presetThreeLayer` с иконкой слоев
- Реализована функция `threeLayerShield` с логикой для 3-слойных экранов
- Включает специфические Constraints:
  - Общая толщина ограничена 30 см
  - Средний слой (тяжелые материалы) должен быть не менее 2 см
  - Каждый слой должен иметь минимальную толщину (0.5 см для легких, 2 см для тяжелых)

### 2. Обновленные существующие пресеты
Все существующие пресеты были переработаны для поддержки 3-слойной структуры:

#### a) Aluminum Shield
- Ранее: толщина + процент алюминия
- Теперь: толщина внешнего слоя (1-5 см), толщина среднего слоя (2-8 см), толщина внутреннего слоя (1-5 см)

#### b) Lead-Concrete Shield
- Ранее: толщина + проценты свинца и бетона
- Теперь: толщина подложки Al (2-10 см), толщина свинца (5-20 см), толщина бетона (10-50 см)

#### c) Multi-Layer Shield
- Ранее: толщина + проценты различных материалов
- Теперь: толщина легкого слоя 1 (1-10 см), толщина тяжелого слоя (2-15 см), толщина легкого слоя 2 (1-10 см)

#### d) Spacecraft Shield
- Ранее: толщина + проценты Al, Li, H
- Теперь: толщина внешнего Al (0.5-3 см), толщина полиэтилена (1-5 см), толщина внутреннего Al (0.5-2 см)

#### e) Reactor Shield
- Ранее: толщина + проценты B, бетона, стали
- Теперь: толщина внутренней стали (5-20 см), толщина бора/бетона (10-30 см), толщина внешнего бетона (20-50 см)

### 3. Улучшения интерфейса
- Обновлен текст описания пресетов, чтобы отражать новую структуру (толщины слоев вместо процентов)
- Добавлена новая кнопка для 3-слойного экрана с иконкой слоев

## Технические детали

### Constraints для 3-слойных экранов
Для пресета `threeLayerShield` реализованы следующие Constraints:

1. **Ограничение общей толщины**: сумма всех слоев ≤ 30 см
2. **Минимальная толщина среднего слоя**: ≥ 2 см (для тяжелых материалов)
3. **Минимальная толщина первого слоя**: ≥ 0.5 см
4. **Минимальная толщина третьего слоя**: ≥ 0.5 см

### Структура матрицы
- Столбцы: отдельные сегменты экрана (разные области защиты)
- Строки: слои экрана (1-й слой, 2-й слой, 3-й слой и т.д.)

## Радиационная защита
Все пресеты теперь соответствуют принципам радиационной защиты:
- Легкие материалы (Al, полиэтилен) для остановки заряженных частиц
- Тяжелые материалы (Pb, W) для ослабления гамма-излучения
- Нейтронные модераторы (бетон, полиэтилен) для защиты от нейтронов

## Заключение
Все пресеты теперь поддерживают 3-слойную структуру экранов с физически обоснованными диапазонами толщин и необходимыми Constraints для эффективной радиационной защиты.
